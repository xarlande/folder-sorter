# ⚡ Folder Sorter CLI

Високопродуктивна консольна утиліта для автоматичного впорядкування файлів, написана на Rust.

---

[🇬🇧 English version](README.md) | [👉 Головна документація](../README_UA.md)

---

## ✨ Особливості

- **Ультрашвидка робота**: Скомпільований бінарний файл Rust для максимальної швидкості I/O операцій.
- **Автоматичне сортування**: Переміщення файлів у підпапки категорій за розширеннями.
- **Гнучка конфігурація**: Налаштування категорій та списків розширень через TOML файл.
- **Обробка дублікатів**: Автоматичне додавання цифрових індексів `filename_1.ext` при збігу імен у цільовій папці.
- **Режим перевірки (Dry Run)**: Можливість переглянути заплановані дії без зміни файлів на диску.

## 🛠️ Збірка та встановлення

Вимоги: [Rust та Cargo](https://rustup.rs/).

```bash
# Збірка release бінарника локально
cargo build --release

# Встановлення бінарника в ~/.cargo/bin
cargo install --path .
```

## 🚀 Використання

```bash
folder-sorter --path <ШЛЯХ_ДО_ПАПКИ> [ОПЦІЇ]
```

### Параметри

- `-p, --path <PATH>` — Шлях до вихідної папки, яку потрібно впорядкувати.
- `-o, --output <OUTPUT>` — *(Опціонально)* Шлях до папки призначення. За замовчуванням — вихідна папка.
- `-d, --dry-run` — Режим перевірки без переміщення файлів.
- `-V, --version` — Показати версію програми.
- `-h, --help` — Показати довідку.

### Приклади

```bash
# Попередній перегляд дій у папці Downloads
folder-sorter --path ~/Downloads --dry-run

# Впорядкувати файли в папці Downloads
folder-sorter --path ~/Downloads

# Вказати окрему папку призначення
folder-sorter --path ~/Downloads --output ~/Organized

# Запуск через Cargo під час розробки
cargo run --release -- --path . --dry-run
```

## ⚙️ Конфігурація

При виконанні програма зчитує файл `cleaner_config.toml` із поточного робочого каталогу. Якщо файл відсутній, вона створює конфігурацію за замовчуванням.

Правила за замовчуванням:
```toml
[rules]
"Зображення" = ["jpg", "png", "jpeg", "gif", "svg"]
"Відео" = ["mp4", "mkv", "mov", "avi"]
"Музика" = ["mp3", "wav", "flac"]
"Документи" = ["pdf", "doc", "docx", "txt"]
"Архіви" = ["zip", "rar", "7z", "tar"]
"Програми" = ["exe", "msi", "deb"]
```

Невпізнані розширення автоматично потрапляють до папки `"Інше"`.
