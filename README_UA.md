# 📁 Folder Sorter

**Folder Sorter** — це сучасний кросплатформений програмний комплекс для автоматичного впорядкування файлів у папках за їхніми розширеннями. Проєкт поєднує високопродуктивне ядро на **Rust (CLI)** та зручний графічний інтерфейс на **Python (Tkinter GUI)** з підтримкою автоматизації за розкладом.

## ✨ Основні можливості

- ⚡ **Висока швидкість (Rust Core)**: Ультрашвидке сканування та переміщення файлів з автоматичним додаванням цифрових індексів при збігу імен.
- 🎨 **Сучасний GUI (Python GUI)**: Темний інтерфейс з окремими вкладками для сортування, налаштування правил та фонового розкладу.
- ⏰ **Автоматизація за розкладом**: Нативна інтеграція з системними планувальниками:
  - **macOS**: `launchd` (LaunchAgents)
  - **Windows**: Task Scheduler (`schtasks`)
  - **Linux**: `cron` (`crontab`)
- 🔍 **Режим перевірки (Dry Run)**: Можливість безпечного перегляду списку запланованих дій без внесення реальних змін на диску.
- ⚙️ **Гнучка конфігурація TOML**: Легке створення власної системи категорій та розширень у файлі `~/.foldersorter/cleaner_config.toml`.

## 🚀 Швидкий запуск

### 1. Вимоги до середовища
- **Rust / Cargo**: потрібен для компіляції CLI ядра ([встановити Rust](https://rustup.rs/)).
- **Python 3.10+** (або [uv](https://docs.astral.sh/uv/)): для запуску GUI застосунку.

### 2. Запуск графічного застосунку (GUI)

За допомогою `uv`:
```bash
cd gui
uv run main.py
```

Або за допомогою стандартного `python3`:
```bash
cd gui
python3 main.py
```

> 💡 *Примітка: GUI автоматично виявляє та компілює бінарний файл Rust CLI у режимі release, якщо він відсутній.*

### 3. Прямий запуск CLI (Rust)

```bash
cd cli

# Запуск у режимі перевірки (Dry Run)
cargo run --release -- --path ~/Downloads --dry-run

# Сортування файлів у вказаній папці
cargo run --release -- --path ~/Downloads
```

## ⚙️ Налаштування конфігурації

Конфігурація зберігається за шляхом `~/.foldersorter/cleaner_config.toml`.

Приклад вмісту `cleaner_config.toml`:

```toml
[rules]
"Зображення" = ["jpg", "png", "jpeg", "gif", "svg"]
"Відео" = ["mp4", "mkv", "mov", "avi"]
"Музика" = ["mp3", "wav", "flac"]
"Документи" = ["pdf", "doc", "docx", "txt"]
"Архіви" = ["zip", "rar", "7z", "tar"]
"Програми" = ["exe", "msi", "deb"]
```

Категорії правил та розширення можна редагувати безпосередньо у файлі або зручно змінювати через вкладку **«Налаштування правил»** у GUI.

## 📦 Збірка автономних інсталяторів (PyInstaller)

Для створення автономних дистрибутивів (`.app` на macOS, `.exe` на Windows або виконуваного файлу для Linux):

```bash
cd gui
python3 build_installer.py
```

Автоматизований скрипт:
1. Скомпілює ядро Rust CLI у режимі `--release`.
2. Упакує Python GUI разом із CLI бінарником у самостійний пакет через PyInstaller.
3. Сформує готові архіви в папці `gui/dist/`.
