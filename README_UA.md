# 📁 Folder Sorter

**Folder Sorter** — це сучасний кросплатформений програмний комплекс для автоматичного впорядкування файлів у папках за їхніми розширеннями. Проєкт поєднує високопродуктивне ядро сортування на **Rust (CLI)** та сучасний графічний десктопний інтерфейс на **Tauri (Rust + Vue 3)**.

---

## ✨ Основні можливості

- ⚡ **Висока швидкість сортування**: Миттєве сканування та прибирання папок з автоматичною обробкою дублікатів файлів.
- 🎨 **Сучасний графічний інтерфейс (Tauri GUI)**: Преміальний темний дизайн на Vue 3 та Tailwind CSS з анімаціями, метриками та інтерактивною консоллю.
- ⏰ **Системне планування**: Автоматичне створення тлових завдань ОС:
  - **macOS**: `launchd` (LaunchAgents)
  - **Windows**: Task Scheduler (`schtasks`)
  - **Linux**: `cron` (`crontab`)
- 🔍 **Режим перевірки (Dry Run)**: Можливість безпечного перегляду списку запланованих дій без внесення реальних змін на диску.
- ⚙️ **Гнучке налаштування правил**: Зручний редактор категорій та розширень файлів із збереженням конфігурації в `~/.foldersorter/cleaner_config.toml`.

## 🚀 Швидкий запуск

### 1. Вимоги до середовища
- **Rust / Cargo**: потрібен для компіляції CLI ядра та Tauri GUI ([встановити Rust](https://rustup.rs/)).
- **Node.js 18+**: потрібен для запуску та збірки фронтенду Tauri.

### 2. Запуск графічного застосунку (Tauri GUI)

```bash
cd tauri-gui
pnpm install
pnpm tauri dev
```

### 3. Прямий запуск CLI (Rust)

```bash
cd cli

# Запуск у режимі перевірки (Dry Run)
cargo run --release -- --path ~/Downloads --dry-run

# Сортування файлів у вказаній папці
cargo run --release -- --path ~/Downloads
```

## ⚙️ Налаштування конфігурації

Конфігурація зберігається за шляхом `~/.foldersorter/cleaner_config.toml`.

Приклад вмісту `cleaner_config.toml`:

```toml
[rules]
"Зображення" = ["jpg", "png", "jpeg", "gif", "svg"]
"Відео" = ["mp4", "mkv", "mov", "avi"]
"Музика" = ["mp3", "wav", "flac"]
"Документи" = ["pdf", "doc", "docx", "txt"]
"Архіви" = ["zip", "rar", "7z", "tar"]
"Програми" = ["exe", "msi", "deb"]
```

Категорії правил та розширення можна редагувати безпосередньо у файлі або зручно змінювати через вкладку **«Налаштування правил»** у GUI.

## 📦 Збірка автономних дистрибутивів

Для створення нативних інсталяторів (`.dmg`/`.app` на macOS, `.msi`/`.exe` на Windows, `.deb`/`.AppImage` на Linux):

```bash
cd tauri-gui
pnpm tauri build
```

Готові збірки будуть сформовані в папці `tauri-gui/src-tauri/target/release/bundle/`.
